# Capturas de Pantalla

Esta carpeta contiene las capturas de pantalla de la aplicación FoodForAll para usar en la documentación y en la tienda de aplicaciones.

## Capturas recomendadas

Para una mejor presentación en Google Play Store, se recomienda incluir:

1. **1-home-map.png** - Pantalla principal con el mapa mostrando restaurantes
2. **2-restaurant-list.png** - Lista de restaurantes con tarjetas
3. **3-search.png** - Pantalla de búsqueda con filtros activos
4. **4-restaurant-profile.png** - Perfil completo de un restaurante
5. **5-create-review.png** - Formulario para crear una reseña
6. **6-user-profile.png** - Perfil de usuario con reseñas y guardados
7. **7-filters.png** - Filtros rápidos de restricciones dietéticas

## Especificaciones para Google Play Store

- **Formato**: PNG o JPEG
- **Tamaño mínimo**: 320px de ancho
- **Tamaño máximo**: 3840px de ancho
- **Relación de aspecto**: 16:9 o 9:16
- **Tamaño de archivo**: Máximo 8 MB por imagen

## Cómo tomar capturas

1. Ejecuta la aplicación en un dispositivo o emulador
2. Navega a la pantalla que deseas capturar
3. Usa el comando de Android Studio: `Tools > Layout Inspector` o toma una captura de pantalla del dispositivo
4. Guarda la imagen en esta carpeta con el nombre correspondiente

## Notas

- Asegúrate de que las capturas muestren datos reales y representativos
- Considera usar diferentes dispositivos (teléfono y tablet) para mostrar adaptabilidad
- Las capturas deben estar en español si la app está dirigida al mercado hispanohablante
